# DevLog Frontend

Frontend React + Vite + Tailwind para o sistema DevLog.

## Setup

1. `cd frontend`
2. `npm install`
3. `npm run dev`

## API

O frontend consome a API Spring Boot em `http://localhost:8080/vagas`.

## Estrutura principal

- `src/main.jsx`
- `src/App.jsx`
- `src/components/Sidebar.jsx`
- `src/components/StatsCards.jsx`
- `src/components/VagasTable.jsx`
- `src/components/VagaModal.jsx`
- `src/services/vagaService.js`
